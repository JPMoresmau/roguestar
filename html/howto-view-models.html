You can make new models in roguestar-gl by writing them directly into the haskell source code.  All models are stored in src/Models/.

To view your model and make sure it looks right, you can follow this example using ghci.  Text that you type in is in bold.  Replace anything saying "ascension class starship" with the name of whatever model you are making.

lane@mr-lambda:~/src/roguestar-gl$ ls
add_to_path.sh  COPYING  _darcs  haddock  header  html  INSTALL  Makefile  products  src
lane@mr-lambda:~/src/roguestar-gl$ cd src/
lane@mr-lambda:~/src/roguestar-gl/src$ ghci
