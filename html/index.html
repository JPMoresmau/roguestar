Information on the roguestar-gl package.

Roguestar-GL is:
  • A OpenGL frontend to the roguestar game engine (roguestar-engine)
  • Free: roguestar-gl is distributed under the GNU GPL.
  • Written in the pure functional programming language haskell.
  • In it's very earliest stages of development.

Installing Roguestar-GL

You can obtain the source code to the latest unstable version of roguestar-gl from this darcs repository.  Binaries of roguestar-gl are NOT available.  Note that roguestar-gl is useless without roguestar-engine.

Once you have darcs installed, downloading roguestar-gl and roguestar-engine is simple:

$ darcs get --partial http://www.downstairspeople.org/darcs/unstable/roguestar-gl
$ darcs get --partial http://www.downstairspeople.org/darcs/unstable/roguestar-engine

Dependencies

To build and run roguestar-gl, you will need the following packages, all of which are available, for example, with Debian GNU/Linux, darwinports (for MacOS X), and probably any other unix-ish distribution.  If you want to get these working on Windows, you're going to want to start with cygwin or mingw and then install the other dependencies by hand.
  • ghc version 6.4.1 or higher
  • netpipes (scroll down to "download")
  • make
  • bash

Steps For Linux/BSD/MacOS X.

Step 1:  Do you have the above dependencies installed?  If not, go back and install them.

Step 2: Change to the roguestar-engine directory and execute the following commands:

$ make
$ su
# make install

Step 3.  Change to the roguestar-gl directory and execute the following commands:

$ make
$ su
# make install

Step 4:  Simply run roguestar-gl.

$ roguestar-gl
