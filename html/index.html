Introduction

Roguestar is:
  • A space adventure role-playing game.
  • Free: distributed under the GNU GPL.
  • Written in the pure functional programming language haskell.
  • In it's very earliest stages of development.

Installing Roguestar-GL

Roguestar is built from two source packages:  roguestar-engine and  roguestar-gl.  You need to download, build, and install both of these packages before roguestar will work.  Binaries and tarballs of roguestar-gl are not currently available.

You can obtain the source code to the latest unstable version of roguestar using darcs.

Once you have darcs installed, downloading roguestar-gl and roguestar-engine is simple:

$ darcs get --partial http://www.downstairspeople.org/darcs/unstable/roguestar-gl
$ darcs get --partial http://www.downstairspeople.org/darcs/unstable/roguestar-engine

Dependencies

To build and run roguestar, you will need the following packages, all of which are available, for example, with Debian GNU/Linux, darwinports (for MacOS X), and probably any other *NIX distro.

A windows port of roguestar-gl should be straightforward, but I don't own a windows computer and don't ever plan to.
  • ghc version 6.4.1 or higher
  • netpipes (scroll down to "download")
  • make
  • bash

Steps For Linux/BSD/MacOS X.

Step 1:  Do you have the above dependencies installed?  If not, go back and install them.

Step 2: Change to the roguestar-engine directory and execute the following commands:

$ make release
$ su
# make install

Step 3.  Change to the roguestar-gl directory and execute the following commands:

$ make release
$ su
# make install

Step 4:  Simply run roguestar-gl.

$ roguestar-gl
